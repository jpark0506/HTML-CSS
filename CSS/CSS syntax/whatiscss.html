CSS는 어떻게 구성되어 있을까? 코드를 이용해서 이해해보자 stylesheet() { selector {declaration block = ...declaration } declaration = {property : value} } 해석: CSS의 구성요소는 stylesheet이다 style sheet는 특정 요소를 가르키는 selector(어떤 요소를 바꿀지 고른다)와 선택한 요소를 어떻게 바꿀지 정의하는 코드블록인 declaration block로 이루어져 있다 declaration block은 declaration의 집합이며 각 declaration은 {property: value} 형식으로 들어있다 property는 특징을 나타내고, value는 선택한 특징을 쓴다
