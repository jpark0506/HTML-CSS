전 세계에서 인터넷 사용을 하는 사람들이 늘고, 인터넷 사용의 편의를 위해서 자신의 원하는 정보를 찾기 위해 검색 엔진을 도입했다
이 검색엔진이 어떻게 작동하는지 알아보면, 인터넷은 로봇이라는 프로그램을 이용해서 정보를 수집한다(이를 크롤링이라고 함)
그리고 사용자가 검색할만한 것을 예상해서 색인을(인덱싱)을 진행한다.(인스타그램 태그를 자동으로 만들어준다고 생각하면 된다)
그런데 인덱싱을 할 때, 로봇이 가져온 정보를 이용해서 인덱싱을 하는 데, 이 과정은 HTML 요소를 이용한다
HTML요소을 이용해서 정보를 인식하는 것은 시멘틱 요소를 이용해서 해석하게 된다
